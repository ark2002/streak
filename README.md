# streak

### Run Locally

1. Clone the project

   ```bash
   git clone https://github.com/ark2002/ark2002-oct-10.git
   ```

2. Go to the project directory

   ```bash
   cd ark2002-oct-10
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the server

   ```bash
   npm run start
   ```
